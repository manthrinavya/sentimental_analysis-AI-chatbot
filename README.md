 🤖 Sentiment Analysis Customer Support Chatbot

A simple customer support chatbot built using **Python** and **Streamlit**. It answers user questions from a CSV dataset and analyzes the sentiment of the user's input.

 Features
- Customer support chatbot
- Sentiment analysis (Positive, Negative, Neutral)
- Chat history
- CSV dataset-based responses
- Simple Streamlit interface

 Technologies
- Python
- Streamlit
- Pandas
- TextBlob

 Run the Project

bash
pip install -r requirements.txt
streamlit run app.py
